Task Manager App

A clean, productivity-focused task manager to create, prioritize, and track tasks with a smooth mobile-first experience.

Problem

Keeping daily tasks organized gets messy when there’s no quick way to add, update, and track progress on the go.

Approach

Built UI in Flutter with reusable components and simple task states. Integrated Firebase for authentication + real-time task storage. Added filters for status and priority to keep the workflow fast.

Result

Functional task manager with add/edit/delete, status-based views, and persistent data sync — designed to feel minimal and quick.
